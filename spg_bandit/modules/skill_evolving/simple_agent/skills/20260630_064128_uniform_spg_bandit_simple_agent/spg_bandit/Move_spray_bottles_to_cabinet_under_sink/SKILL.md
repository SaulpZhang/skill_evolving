---
name: Move_spray_bottles_to_cabinet_under_sink
description: This skill moves all spray bottles from countertops to the cabinet located directly under the sink. Use when the task requires storing spray bottles under the sink.
---

1. Find all countertops with spray bottles (e.g., countertop 1) and take each spray bottle. 2. Locate the sinkbasin (e.g., sinkbasin 1) and then identify the cabinet directly below it (often named "cabinet_under_sink" or similar). 3. Open that cabinet. 4. Place each spray bottle inside the cabinet.
